# CuteNotepad 📝✨

A beautiful, modern, and delightful notepad application.

## Features
- **Charming Design**: Aesthetic user interface with rich visual design, smooth animations, and soft color palettes.
- **Auto-save**: Never lose your thoughts or notes.
- **Rich formatting**: Text formatting, emoji support, and easy organization.

## Tech Stack
- HTML5, CSS3, and JavaScript

## Getting Started
To run this application locally, simply open the `index.html` file in a web browser, or start a local dev server.
